The Modal feature facilitates the opening and closing of a modal window from any non circular reference component/service in the application via the ModalService.

  • The modal feature supports the following payload options
    • String
    • Template reference
    • Component reference
  • Multiple opened modals are supported
    • Modals are opened in a stack fasion (Last one opened is "on top")
    • Multiple modals are closed in the reverse order that they were opened (The last one opened is closed first)
  • Modals can be closed by:
    • Clicking the "x" in the top right corner
    • Using the "OK" button if it is present
    • Using the "Close" button if it is present
    • Using the "ESC" key
    • Using a button supplied by a component opened as a modal with custom code
  • Ok, Close and Cancel buttons are overridable

The purpose of this tab is to provoide you with an expectation of the behavior of the modal feature. Implementation or usage of the modal feature can be found in gw-lib/core/modal/src/modal.service.ts.
